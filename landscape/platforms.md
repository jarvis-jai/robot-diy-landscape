## 條目

### Hackster.io

- **Name**：Hackster.io
- **Type**：Project Hub
- **Primary language**：English
- **一句話定位**：Avnet 旗下的硬體專案分享與學習社群，涵蓋 IoT、機器人、AI 等領域
- **強項（≤3）**：
  - 完整的專案分享平台，支援影片、步驟說明、BOM 表
  - 有 Channels 系統依主題分類（Robotics、IoT、AI & Machine Learning、Home Automation 等）
  - 定期舉辦 Contests 與 Events，鼓勵社群參與
- **弱項/缺口（≤3，可觀察）**：
  - 無中文介面或內容分類
  - 部分專案文件不完整（僅有概念，缺少詳細步驟）
  - 需登入才能下載完整資源
- **活躍度訊號**：持續更新中（2026-01 有 CES 2026 報導、Arduino UNO Q 評測等新內容）；有活躍的 YouTube 頻道
- **與我們的關聯（一句話）**：機器人 DIY 專案的主要英文資源庫，可發現熱門專案與硬體趨勢
- **Link(s)**：https://www.hackster.io/

### Instructables

- **Name**：Instructables
- **Type**：Project Hub
- **Primary language**：English
- **一句話定位**：Autodesk 旗下的 DIY 專案分享平台，涵蓋電子、木工、烹飪等多元類別
- **強項（≤3）**：
  - 20 年歷史（2006 年創立），內容豐富且分類完整
  - 有 Circuits 類別專門收錄電子與機器人專案
  - 有 Teachers 專區提供教育資源，適合教學場景
- **弱項/缺口（≤3，可觀察）**：
  - 內容範圍廣泛但機器人專案比例較低
  - 無中文介面
  - 部分舊專案使用過時的零件或技術
- **活躍度訊號**：持續有新專案上傳（2026-01）；有 Instagram、TikTok 社群帳號
- **與我們的關聯（一句話）**：適合入門者的 DIY 教學平台，可找到基礎機器人專案教程
- **Link(s)**：https://www.instructables.com/

### Arduino Project Hub

- **Name**：Arduino Project Hub
- **Type**：Project Hub
- **Primary language**：English
- **一句話定位**：Arduino 官方的專案分享平台，專注於 Arduino 生態系統的專案
- **強項（≤3）**：
  - 官方平台，與 Arduino 產品線深度整合
  - 有明確的分類系統（Robotics、IoT、Home Automation、Flying Things 等）
  - 支援難度分級（Tutorial / Featured / Pro / For School）
- **弱項/缺口（≤3，可觀察）**：
  - 僅限 Arduino 生態，不涵蓋 ROS 或其他進階框架
  - 無中文介面
  - 專案數量較 Hackster.io 少（約 5,956 個專案）
- **活躍度訊號**：持續更新，有 Trending 排序功能顯示近期熱門專案
- **與我們的關聯（一句話）**：Arduino 入門機器人專案的首選資源庫
- **Link(s)**：https://projecthub.arduino.cc/

### Arduino Cloud

- **Name**：Arduino Cloud
- **Type**：Cloud Platform
- **Primary language**：English
- **一句話定位**：Arduino 官方的雲端開發與 IoT 管理平台
- **強項（≤3）**：
  - 提供瀏覽器內的 IDE，無需本地安裝即可開發
  - 支援遠端設備監控與控制（IoT Dashboard）
  - 整合 Google、GitHub、Apple 登入
- **弱項/缺口（≤3，可觀察）**：
  - 免費方案功能有限，進階功能需付費訂閱
  - 依賴網路連線，離線開發受限
  - 無中文介面
- **活躍度訊號**：官方持續維護，與 Arduino 硬體新品同步更新
- **與我們的關聯（一句話）**：適合快速原型開發的雲端工具，降低入門門檻
- **Link(s)**：https://app.arduino.cc/

### Thingiverse

- **Name**：Thingiverse
- **Type**：3D Model Repository
- **Primary language**：English
- **一句話定位**：最大的免費 3D 打印模型分享平台，由 MakerBot（現 UltiMaker）運營
- **強項（≤3）**：
  - 龐大的模型庫，涵蓋機器人零件、外殼、機械結構
  - 支援 Makes 功能，可查看他人的實際打印成果
  - 免費下載 STL 文件
- **弱項/缺口（≤3，可觀察）**：
  - 近年用戶體驗下降，網站性能問題頻繁被反映
  - 搜尋功能不夠精確
  - 無中文介面
- **活躍度訊號**：仍為主流平台（Popular 專案有 1K+ likes），但用戶流失至 Printables
- **與我們的關聯（一句話）**：機器人 DIY 零件與外殼的 3D 打印資源庫
- **Link(s)**：https://www.thingiverse.com/

---

### Printables

- **Name**：Printables
- **Type**：3D Model Repository
- **Primary language**：English
- **一句話定位**：Prusa Research 運營的 3D 打印模型平台，Thingiverse 的主要競爭者
- **強項（≤3）**：
  - 現代化的使用者介面，搜尋與瀏覽體驗佳
  - Prusameters 積分系統激勵高品質貢獻
  - 整合 EasyPrint 線上切片功能
- **弱項/缺口（≤3，可觀察）**：
  - 模型總量仍少於 Thingiverse
  - 偏向 Prusa 用戶群，其他品牌打印機支援次之
  - 無中文介面
- **活躍度訊號**：快速成長中，定期舉辦 Contests（如 Valentine's Classics 2026）
- **與我們的關聯（一句話）**：高品質 3D 打印資源的新興平台，機器人零件模型持續增加
- **Link(s)**：https://www.printables.com/

### GrabCAD

- **Name**：GrabCAD
- **Type**：CAD Model Library
- **Primary language**：English
- **一句話定位**：Stratasys 旗下的專業 CAD 文件分享社群，面向工程師與設計師
- **強項（≤3）**：
  - 提供專業級 CAD 文件（SolidWorks、Fusion 360、STEP 等格式）
  - 有 Challenges 競賽促進創新設計
  - 社群功能完整（Groups、Questions、Tutorials）
- **弱項/缺口（≤3，可觀察）**：
  - 偏向專業工程領域，入門者較難上手
  - 部分模型需專業 CAD 軟體才能編輯
  - 無中文介面
- **活躍度訊號**：持續運營，有活躍的社群問答與教學資源
- **與我們的關聯（一句話）**：進階機器人設計的 CAD 資源庫，適合需要精密機械設計的專案
- **Link(s)**：https://grabcad.com/library

### Hackaday.io

- **Name**：Hackaday.io
- **Type**：Project Hub
- **Primary language**：English
- **一句話定位**：Hackaday 旗下的開源硬體專案平台，聚焦創新與駭客精神
- **強項（≤3）**：

  - 專案難度偏高，不適合純新手
  - 介面較舊，使用體驗不如新平台
  - 無中文介面
- **活躍度訊號**：持續有新專案發布（2026-01 有機器人手臂、農業機器人等活躍專案）
- **與我們的關聯（一句話）**：開源機器人專案的深度技術資源，適合進階 DIY 者
- **Link(s)**：https://hackaday.io/
